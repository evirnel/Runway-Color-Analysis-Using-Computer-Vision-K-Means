# Runway-Color-Analysis-Using-Computer-Vision-K-Means
Breaking down Ashlynn Park's Fall 2026 opening look using DeepLabV3 for segmentation, OpenCV for skin/head removal, and K-Means clustering to extract the core color palette. The result? Black (70.6%), Cream (16.9%), and Citron Yellow (12.5%) , revealing the true color story behind the first look.
